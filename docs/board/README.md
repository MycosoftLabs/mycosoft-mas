# Board Materials

**Updated:** 2026-03-05

Board packages and meeting materials are generated here. MYCA uses `config/board_meetings.yaml` to schedule and produce board packages.

## Contents

- Board packages: `board_package_YYYY-MM-DD.md` (generated per meeting)
- Minutes: stored after each meeting

## Config

See `config/board_meetings.yaml` for meeting dates, agenda, and compliance settings.

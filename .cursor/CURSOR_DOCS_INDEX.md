# Cursor Docs Index – Internal Reference (Vital & Current Only)

**Policy:** This file is the canonical list of **vital and current** docs for Cursor agents. It is updated continuously: **new docs replace old ones** in the list. Do not add deprecated or superseded docs; when a new doc replaces an old one, remove the old entry.

**Use:** Before any plan or implementation, read this index then open the listed docs as needed. For **full discovery** of all 2000+ docs (including older/superseded), use `.cursor/docs_manifest.json` (generated by `python scripts/build_docs_manifest.py`) or `docs/MASTER_DOCUMENT_INDEX.md`.

**Why Cursor doesn’t “see” all 2000+ docs automatically:** Cursor does not load every file in the workspace into context. Context window is limited; agents must use this index + manifest + semantic search to find and open relevant docs. **Notion** is a one-way sync target (docs pushed to Notion for humans); Cursor only sees **local files** in the workspace.
